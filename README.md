# Air Fighter

A simple 2D game in C using SDL2

## OVERVIEW

This is just a small project to get myself familiar with SDL, C and game dev concepts
such as:
- Game loops
- Input handling
- Collision Detection
- Audio play
- Rendering

## features inclede:

- Keyboard controls
- Sprite-based rendering
- Sound effects and Music
- Score tracking
- Pause Menu

## Tech used:

- C
- SDL2
-SDL2_IMAGE
-SDL2_MIXER
